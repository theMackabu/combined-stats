## account stats combiner

#### made for the [bloodlust guild](https://discord.com/invite/bloodlust)

##### install steps

- copy .example.env to .env
- update all fields in .env
- `yarn`
- `yarn build`
- `yarn start`

congratulations, you have bot running now! (hopefully)
